# Animal Isle 🍃

A cozy Animal Crossing-style fan demo in a single self-contained HTML file. No build, no dependencies (just Google Fonts).

**Play it:** https://fabiosanchez-aircury.github.io/animal-crossing-web/

- Real 3D renderer (three.js / WebGL): smooth heightmap terrain, rounded characters, dynamic sun + shadows, drag to orbit, wheel to zoom
- Boat travel to biome islands (tropical, glacial, volcano) and a floating market, each with exclusive resources
- Combat (sword vs. magma slimes), fishing minigame with 10 species, rock mining and fossil digging with animations
- Building interiors with loading screens, museum with identifiable/donatable collections on display, clothes shop + wardrobe customization
- Real-time day/night cycle and seasons synced to your local clock
- Fully procedural audio via Web Audio API (generative music + birds, crickets, wind, waves)
- Procedural island per save seed · 3 save slots (localStorage)
- Progression: tools → house → shop → museum → café → new neighbors
- English / Español (press L)

Older versions: `v1.html` (flat 2D), `v2.html` (2.5D rolling world), `v3.html` (first voxel 3D).

Fan-made demo for fun — not affiliated with Nintendo.
