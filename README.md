# Animal Isle 🍃

A cozy Animal Crossing-style fan demo in a single self-contained HTML file. No build, no dependencies (just Google Fonts).

**Play it:** https://fabiosanchez-aircury.github.io/animal-crossing-web/

- Real 3D voxel-style renderer (three.js / WebGL): dynamic sun and shadows, procedural textures, drag to orbit the camera, wheel to zoom
- Real-time day/night cycle and seasons synced to your local clock
- Fully procedural audio via Web Audio API (generative music + birds, crickets, wind, waves)
- Procedural island per save seed · 3 save slots (localStorage)
- Progression: tools → house → shop → museum → café → new neighbors
- English / Español (press L)

`v1.html` is the original flat top-down version; `v2.html` is the 2.5D "rolling world" version.

Fan-made demo for fun — not affiliated with Nintendo.
